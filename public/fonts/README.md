# Font Files

Place KongText font files here:

- `KongText-Regular.woff2`
- `KongText-Regular.woff`
- `KongText-Regular.ttf`

If you don't have the KongText font, the app will fall back to `Courier New` monospace font.

## Download KongText

KongText is a pixelated retro font. You can find it at:
- [Google Fonts](https://fonts.google.com/) (if available)
- Font repositories or design resources

Alternatively, you can use any pixelated monospace font and update the `@font-face` declaration in `src/index.css`.
